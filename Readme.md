# OIBSIP_DataAnalytics_Level1_Project1

## Exploratory Data Analysis on Retail Sales Data using Python

### Project Overview
This project performs Exploratory Data Analysis (EDA) on a retail sales dataset to uncover sales trends, customer purchasing behavior, and product performance.

The analysis focuses on identifying:
- Top-selling products
- Best-performing branches
- Revenue drivers
- Product line performance
- Sales relationships and correlations

---

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## Dataset

Retail Sales Dataset (Source: Kaggle)

The dataset contains transaction-level information including:
- Date
- Branch
- Product Line
- Quantity
- Unit Price
- Tax
- Total Sales Amount

---

## Analysis Performed

- Data Loading and Cleaning
- Descriptive Statistics
- Branch-wise Sales Analysis
- Product Line Sales Analysis
- Sales vs Quantity Correlation Analysis
- Data Visualization

## Key Insights

- Branch C generated the highest sales revenue.
- Food and Beverages was the top-performing product category.
- Strong positive correlation exists between Quantity and Total Sales.
- Product line performance varied across branches.

---

## Recommendations

- Increase inventory for high-performing product categories.
- Develop promotional strategies for low-performing products.
- Replicate successful sales practices across all branches.
- Focus marketing efforts on top-selling product lines.

---

## Conclusion

The analysis revealed important sales patterns and product performance trends. These insights can help businesses improve inventory planning, optimize marketing campaigns, and enhance overall sales performance.
